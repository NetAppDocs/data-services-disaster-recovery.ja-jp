= NetAppディザスタリカバリに関するよくある質問
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
質問に対する簡単な回答を探している場合は、この FAQ が役立ちます。

* NetAppディザスタリカバリ URL とは何ですか?*  URL については、ブラウザで次のように入力します。 https://console.netapp.com/["https://console.netapp.com/"^] NetAppコンソールにアクセスします。

* NetApp Disaster Recovery を使用するにはライセンスが必要ですか?*完全なアクセスには、 NetApp Disaster Recovery ライセンスが必要です。ただし、無料トライアルで試してみることはできます。

NetApp Disaster Recoveryのライセンス設定の詳細については、link:../get-started/dr-licensing.html["NetApp Disaster Recoveryライセンスを設定する"] 。

* NetApp Disaster Recovery にどのようにアクセスしますか?* NetApp Disaster Recovery では有効化は必要ありません。災害復旧オプションは、 NetAppコンソールの左側のナビゲーションに自動的に表示されます。
